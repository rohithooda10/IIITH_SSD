### Solution
1. Install express, express-session, mongoose, dotenv, nodemon, monogdb, core, react js,react-router-dom, node js before running the project.
2. Go into server folder and run command npm install to install node_modeules.
3. Go into client/frontend folder and run command npm install to install node_modeules.
4. React server will begin on PORT 3000, and node js server will run on PORT 3001.
5. To start react app, run command npm start in client/frontend.
6. To start backend server, run command nodemon server.js inside the server folder.
